# TRINETRA AI - Quick Start Demo Guide

## 🚀 5-Minute Setup to Demo-Ready

### Step 1: Start the System (2 minutes)
```bash
# Navigate to project directory
cd /path/to/trinetra-ai

# Start the system
python main.py

# Wait for these messages:
# ✅ "FastAPI server started on http://localhost:8000"
# ✅ "Dashboard available at http://localhost:8501"
```

### Step 2: Verify System (1 minute)
1. Open browser: http://localhost:8501
2. Dashboard should load in ~2 seconds
3. Check that KPIs show: 1,000 transactions, 11.7% fraud rate
4. Scroll down - verify all sections visible

### Step 3: Test AI (1 minute)
1. Scroll to "Suspicious Transactions" table
2. Find transaction TXN00006
3. Click "Get AI Explanation" (if available)
4. Verify explanation appears (or fallback message)

### Step 4: Open Demo Materials (1 minute)
1. Open `examples/DEMO_WALKTHROUGH.md` (your main script)
2. Have `examples/demo_scenarios.json` accessible
3. Keep `docs/DEMO_QUICK_REFERENCE.md` handy

### ✅ You're Demo-Ready!
