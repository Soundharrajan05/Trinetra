# 🛡️ TRINETRA AI - Executive Summary

## The Problem

**$1.6 trillion** is lost annually to international trade fraud. Traditional detection methods are:
- ❌ Manual and time-consuming
- ❌ Expensive to operate
- ❌ Miss complex fraud patterns
- ❌ Discover fraud too late

## Our Solution

**TRINETRA AI** - An intelligent trade fraud detection system that combines:

🤖 **Machine Learning** - Automated anomaly detection using IsolationForest  
🧠 **Generative AI** - Natural language fraud explanations via Google Gemini  
📊 **Interactive Visualization** - Real-time dashboards and geographic mapping  
🚨 **Smart Alerts** - Automated notification of high-risk transactions

## Key Features

### 1. Automated Fraud Detection
- Analyzes 1,000+ transactions in seconds
- Classifies risk: SAFE, SUSPICIOUS, FRAUD
- 6 engineered fraud-detection features
- No labeled training data required

### 2. AI-Powered Explanations
- Natural language fraud analysis
- Explains WHY transactions are suspicious
- Helps investigators understand patterns
- Quota management with fallback system

### 3. Visual Intelligence
- Route Intelligence Map - Geographic trade visualization
- Price Deviation Chart - Pricing anomaly detection
- Company Risk Network - Entity relationship mapping
- Interactive dashboards with real-time KPIs

### 4. Investigation Tools
- Natural language query interface
- Ask questions like "Why is this transaction suspicious?"
- Context-aware AI responses
- Transaction detail exploration

## Technical Highlights

**Architecture:** FastAPI backend + Streamlit frontend + Gemini AI  
**ML Algorithm:** IsolationForest (unsupervised learning)  
**Deployment:** Single command (`python main.py`)

**Testing:** >80% code coverage, property-based tests
